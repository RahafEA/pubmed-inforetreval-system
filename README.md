
# PubMed IR System

Information Retrieval system for cancer-related PubMed articles.

## Features
- Fetches articles from PubMed
- Boolean search (AND / OR)
- Vector Space Model with TF-IDF ranking
- Evaluation metrics (MAP, Precision, Recall, F1)

## How to run
1. Install requirements:
pip install -r requirements.txt

2. Run the GUI:
python simple_tkinter_gui_starter.py
